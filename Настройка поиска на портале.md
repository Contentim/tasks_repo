Задать Виджет-мишень - связка между модулем поиска и универсальным списком. Основная идея - виджет-мишенью указать на какую выборку ориентироваться поисковой строке.

Затем - настроить конфиг, прописав строки:
```
{ "search": ["name","desc"],
"columns":[["search"]],
"filters": [ ] }
```
