oSort - информация из рантайма о сортировке (объект). Объект включает в себя следующие данные: FIELD - название поля, по которому производится сортировка (строка), DIRECTION - направление сортировки (строка) (допустимые значения:  "ASC" (сортировка по возрастанию), "DESC" (сортировка по убыванию)).

 oPaging - информация из рантайма о пейджинге (объект). Объект включает в себя следующие данные:  
 INDEX - номер страницы (целое число);  
 SIZE - размер страницы (целое число); 
 MANUAL - разделение на страницы вручную (булево); 
 TOTAL - общее количество страниц (целое число). 
 START_INDEX - начальный элемент списка.
 
Параметры объектов oSort и oPaging можно задавать в формате JSON. Примеры из программных файлов Websoft HCM: 
oParams.oSort = {"FIELD": "person_id", "DIRECTION": "ACS"};
oPaging = { index: iPageIndex, size: iPageSize }
