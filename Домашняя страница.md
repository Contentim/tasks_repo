# **WebSoft**
```base
filters:
  or:
    - file.hasTag("websoft")
    - and:
        - file.hasTag("помощь")
formulas:
  formatted_price: if(price, price.toFixed(2) + " dollars")
  ppu: (price / age).toFixed(2)
properties:
  status:
    displayName: Status
  formula.formatted_price:
    displayName: Price
  file.ext:
    displayName: Extension
  file.ctime:
    displayName: ctime
  file.mtime:
    displayName: mtime
views:
  - type: table
    name: WebSoft
    order:
      - file.name
      - file.ext
      - file.mtime
      - file.ctime
    sort:
      - property: file.mtime
        direction: DESC
      - property: file.ctime
        direction: ASC
      - property: age
        direction: DESC
    limit: 5
    columnSize:
      file.mtime: 156
      file.ctime: 123
  - type: table
    name: представление

```
