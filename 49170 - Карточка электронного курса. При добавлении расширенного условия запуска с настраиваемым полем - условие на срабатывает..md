пример - используем person_id чтобы отсечь запуск этапа курса. за пример взять Анисимов

если попытаться редактировать запрос, то получим строку вида :
```
XQueryLiteral( XQueryLiteral( OptInt(person_id,null) ) )==XQueryLiteral( OptInt('0x0F572070E0E1C121') )
```
 
для того, чтобы запрос работал, он должен выглядеть так:
```
OptInt(curObject.person_id,null) == OptInt(1105387902724063510)
```
ключевое: curObject перед person_id. Димоновский алгоритм не позволяет приклеить во фрагменте OptInt(person_id,null) строку curObject.
